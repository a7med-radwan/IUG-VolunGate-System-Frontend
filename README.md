# IUG VolunGate System Frontend / الواجهة الأمامية لنظام بوابة التطوع

## English

This repository contains the frontend interfaces for the IUG VolunGate system. It includes dashboards and management pages for the Admin, Supervisor, and Student (Volunteer) roles.

### Features
- **Admin Dashboard**: Manage users, roles, and system settings.
- **Supervisor Dashboard**: Manage tasks, monitor student activities, and generate reports.
- **Student Dashboard**: Browse tasks, apply for volunteer opportunities, and track volunteer hours.

---

## العربية

يحتوي هذا المستودع على واجهات المستخدم الأمامية لنظام بوابة التطوع. يتضمن لوحات التحكم وصفحات الإدارة لأدوار المسؤول (Admin)، المشرف (Supervisor)، والطالب/المتطوع (Student).

### المميزات
- **لوحة تحكم المسؤول**: إدارة المستخدمين والأدوار وإعدادات النظام.
- **لوحة تحكم المشرف**: إدارة المهام ومراقبة أنشطة الطلاب وإنشاء التقارير.
- **لوحة تحكم الطالب**: تصفح المهام، التقدم لفرص التطوع، وتتبع ساعات التطوع.
